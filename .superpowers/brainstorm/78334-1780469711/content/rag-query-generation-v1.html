Query Strategies
D

Direct Search

默认路径，直接 embedding 查询。稳定、低成本，适合作为 MVP。

H

HyDE

先生成假设答案再检索。适合抽象问题，但会增加一次 LLM 调用。

M

Multi Query

生成多个子查询后合并检索。适合复杂问题，但要做去重和分数归一。
