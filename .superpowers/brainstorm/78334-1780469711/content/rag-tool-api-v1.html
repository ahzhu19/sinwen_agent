RagTool API 与错误处理

Tool 是 Agent 入口，底层调用 RagManager；返回文本要对人可读，也要方便 Agent 继续推理。

Actions
ingest
摄取文件/URL
search
只检索 chunks
answer
检索 + 生成
delete
删除文档和向量
stats
知识库统计
list_documents
文档列表
reindex
重建向量索引
Failure Modes

可恢复

  • 转换失败：记录 ingestion_run error
  • 向量写入失败：文档和 chunk 保留为 indexed=false
  • 生成失败：仍返回 search sources 供调试

需要明确报错

  • MarkItDown 未安装
  • Embedding 配置缺失
  • PostgreSQL / Milvus 不可用
